# Video Sharing site

This is a vid sharing site rebuilt from the ground up using modern systems to replace the original vshare site.

## Dependancies

This is written for the following:

- server
    - apache2 v2.4.52
    - php-fpm v8.1
    - mariadb 10.3.37
    - s3fs
    - ffmpeg 4.4.2 with ffprobe


- code
    - bootstrap 5.3
    - jquery 3.5.1
    - tagify (latest)
    - recaptcha (latest)
    - ovenplayer (latest)
